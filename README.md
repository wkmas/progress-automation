# progress-automation

個人開発の進捗管理を自動化するためのリポジトリ。

## LLM 選定ログ（2026-05-10）

### 現状の選定: Gemini API（無料枠）

Phase 2 の AI 機能（コミット要約・TODO 生成・週次レポート）に使用する LLM として、
Google Gemini API の無料枠（Gemini 2.5 Flash）を採用。

**選定理由:**
- 無料枠で 250 RPD / 10 RPM が利用可能。現在の用途（日 15 リクエスト程度）に十分
- クレジットカード不要で開始可能
- コミット要約・TODO 生成程度のタスクなら Flash モデルで品質十分

**注意事項:**
- 無料枠では Google がプロンプト・レスポンスをモデル改善に使用する可能性あり（機密情報を送らないこと）
- 無料枠のレート制限は Google 側の判断で変更される可能性あり（2025年12月に50-80%削減の前例あり）

**チーム展開時の移行方針:**
- チーム利用で日次リクエストが 100 回を超える見込みなら、以下を検討:
  1. Gemini API 有料枠（Tier 1: 課金有効化のみ、従量課金）
  2. Claude API（Sonnet 4.6: $3/$15 per MTok、品質重視の場合）
  3. OpenAI API（GPT-4o: $2.50/$10 per MTok）
- LLM 呼び出しは `scripts/llm.js` に抽象化されているため、プロバイダー切り替えは設定変更のみで可能

**切り替え手順:**
1. `scripts/llm.js` の `LLM_PROVIDER` を変更
2. 対応する API キーを GitHub Secrets に追加
3. 動作確認
